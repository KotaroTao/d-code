# D-Code Academy インフラ構成検討

## 結論：Vercel + Supabase（推奨）

年間300名規模の進捗管理・個別サポートシステムであれば **Vercel + Supabase で十分対応可能**です。GCPは現段階ではオーバースペックです。

---

## 比較表

| 要件 | Vercel + Supabase | GCP |
|------|-------------------|-----|
| **進捗管理ダッシュボード** | ✅ Next.js + Supabaseで構築可能 | ✅ 可能だが構築コスト大 |
| **受講者データ管理（300名/年）** | ✅ Supabase無料枠でも余裕 | ✅ Cloud SQL等で可能 |
| **リアルタイム更新** | ✅ Supabase Realtimeで対応 | ✅ Firestore等で可能 |
| **認証・ログイン** | ✅ Supabase Auth | ✅ Firebase Auth / Identity Platform |
| **ファイルストレージ** | ✅ Supabase Storage | ✅ Cloud Storage |
| **構築の容易さ** | ✅ Claude Codeで即構築 | ⚠️ 設定・権限管理が複雑 |
| **運用コスト** | ✅ 月額 $0〜25程度 | ⚠️ 月額 $30〜100+（規模による） |
| **スケーラビリティ** | ✅ 300名/年なら問題なし | ✅ 大規模にも対応 |
| **サーバーレス関数** | ✅ Vercel Functions | ✅ Cloud Functions |
| **学習コスト** | ✅ 低い | ⚠️ 高い（IAM、ネットワーク設定等） |

## Vercel + Supabase で構築するシステム構成

```
受講者 / 講師
    │
    ▼
┌─────────────────────────┐
│  D-Code Academy Web App │  ← Vercel (Next.js)
│  - 講座コンテンツ        │
│  - 進捗管理ダッシュボード │
│  - 個別チャット/Q&A     │
│  - 課題提出システム       │
└──────────┬──────────────┘
           │
           ▼
┌─────────────────────────┐
│      Supabase            │
│  - PostgreSQL (データ)   │
│  - Auth (認証)           │
│  - Realtime (リアルタイム)│
│  - Storage (ファイル)    │
│  - Edge Functions        │
└─────────────────────────┘
```

## 進捗管理・個別サポートシステムの機能設計

### 1. 受講者ダッシュボード
- Day 1〜5の進捗バー表示
- 課題の提出状況（未提出 / 提出済 / レビュー完了）
- デプロイURLの登録・共有
- 個別メモ・質問の記録

### 2. 講師管理画面
- 全受講者の進捗一覧（一目で遅れている人を把握）
- 課題のレビュー・フィードバック入力
- 受講者ごとのサポート履歴
- コホート（期ごと）の管理

### 3. 個別サポート機能
- 受講者↔講師のQ&Aスレッド（Supabase Realtimeで即時反映）
- よくある質問のFAQデータベース
- スクリーンショット添付（Supabase Storage）

### 4. 課題管理
- Day別の課題定義
- GitHubリポジトリURL / VercelデプロイURLの提出
- 講師によるコードレビューコメント

## GCPに移行すべきタイミング

以下の条件に該当する場合はGCPへの移行を検討してください：

| 条件 | 理由 |
|------|------|
| 受講者が年間1,000名を超える | DBの同時接続数・パフォーマンス |
| 動画配信を自前で行う | Cloud CDN + Cloud Storageが有利 |
| 複雑なバッチ処理が必要になる | Cloud Run / Cloud Functionsの柔軟性 |
| 企業向けにSLA保証が必要 | GCPのエンタープライズサポート |
| 他のGCPサービス（AI/ML等）と連携 | エコシステムの統一 |

## コスト見積もり（年間300名規模）

### Vercel + Supabase

| サービス | プラン | 月額 |
|----------|--------|------|
| Vercel | Pro | $20 |
| Supabase | Pro | $25 |
| ドメイン | d-code.com | 〜$1（年額を月割） |
| **合計** | | **〜$46/月（約7,000円）** |

### GCP（参考）

| サービス | 構成 | 月額 |
|----------|------|------|
| Cloud Run | 最小構成 | $10〜30 |
| Cloud SQL | db-f1-micro | $10〜30 |
| Cloud Storage | 標準 | $1〜5 |
| Firebase Auth | 無料枠内 | $0 |
| ロードバランサ | 基本 | $20〜 |
| **合計** | | **〜$50〜100/月** |

※ GCPは構築・運用の人的コストも加味すると、さらに差が広がります。

## 推奨アクションプラン

1. **Phase 1（講座開始前）**: Vercel + Supabaseで進捗管理システムのMVPを構築
2. **Phase 2（初回講座後）**: 受講者フィードバックを元に機能改善
3. **Phase 3（年間100名達成後）**: パフォーマンス評価、必要に応じてSupabaseプランのアップグレード
4. **Phase 4（年間1,000名超）**: GCP移行を検討（この段階で初めて検討すれば十分）
