# D-Code Academy

Claude Code完全初心者向け講座プログラム

## 講座概要

| 項目 | 内容 |
|------|------|
| 対象 | プログラミング・Claude Code完全初心者 |
| ゴール | 自分が作りたいアプリを完成させる |
| 期間 | 3時間 × 5日 = 15時間（各日15分休憩含む） |
| 環境 | Claude Code WEB版 → Vercel自動デプロイ（ローカルファイル不要） |
| URL | https://www.d-code.com/academy/ （独自ドメインは最終日に設定） |
| 想定受講者数 | 年間約300名 |

---

## Day 1: イントロダクション & Claude Codeの基礎（3時間）

### セッション1（80分）：バイブコーディングとは何か

| 時間 | 内容 |
|------|------|
| 0:00 - 0:15 | **オリエンテーション** — 自己紹介、講座の全体像、ゴール設定シートの記入 |
| 0:15 - 0:35 | **バイブコーディング概論** — 従来のプログラミングとの違い、AIと協働する新しい開発スタイル |
| 0:35 - 0:55 | **Claude Codeとは** — 機能紹介、他のAIコーディングツールとの比較、できること/できないこと |
| 0:55 - 1:20 | **デモ：Claude Codeで「Hello World」サイトを5分で作る** — 講師のライブデモ＋受講者が一緒に体験 |

### 休憩（15分）

### セッション2（85分）：環境構築とはじめての開発

| 時間 | 内容 |
|------|------|
| 1:35 - 1:55 | **アカウント準備** — Claude Code WEB版、GitHub、Vercelアカウントの作成・連携 |
| 1:55 - 2:20 | **GitHubの超基礎** — リポジトリとは、なぜ必要か、ブランチの概念（ローカル操作は不要と強調） |
| 2:20 - 2:45 | **ハンズオン①：はじめてのプロジェクト作成** — Claude Codeで自己紹介ページを作成、GitHubにpush |
| 2:45 - 3:00 | **Vercelデプロイ体験** — GitHub連携で自動デプロイ、自分のサイトが公開される感動体験 |

### Day 1 課題
- 自己紹介ページに趣味・好きなものの情報を追加してデプロイ

## 講座で使用する技術スタック

| カテゴリ | ツール | 用途 |
|----------|--------|------|
| AI開発環境 | Claude Code WEB版 | コード生成・編集・デバッグ |
| バージョン管理 | GitHub | コード管理・Vercel連携 |
| フレームワーク | Next.js (React) | Webアプリ構築 |
| デプロイ | Vercel | 自動デプロイ・ホスティング |
| データベース | Supabase | データ保存・認証 |
| スタイリング | Tailwind CSS | UIデザイン |

## 受講者が持参するもの

- ノートPC（ブラウザが動けばOK。スペック不問）
- Googleアカウント（各種サービス登録用）
- 作りたいアプリのアイデア（ぼんやりでOK）
