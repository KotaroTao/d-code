# サポート人員 募集方法 提案書

## エグゼクティブサマリー

D-Code Academy の2コース体制（導入コース/本格運用コース）に対応するため、
合計8〜11名のサポート人員を確保する必要がある。
本提案書では、募集チャネル・タイムライン・選考フロー・コスト見積もりを整理し、
最適な募集戦略を提案する。

---

## 1. 募集対象と優先順位

### フェーズ1：最優先（4月中に確保）

| ポジション | 人数 | 理由 |
|-----------|------|------|
| 導入コース TA | 3〜4名 | 6月開講の主力。最も人数が必要 |
| 運営スタッフ | 1名 | 両コース兼任可能。早期に確保して準備に参加してもらう |

### フェーズ2：準優先（5月中に確保）

| ポジション | 人数 | 理由 |
|-----------|------|------|
| 本格運用コース メイン講師 | 1名 | GCP経験者は希少。時間をかけて質の高い人材を確保 |
| 本格運用コース TA | 2〜3名 | GCP経験が必要なため、導入コースTAより採用難易度が高い |

### フェーズ3：状況次第

| ポジション | 人数 | 理由 |
|-----------|------|------|
| 導入コース メイン講師 | 1名 | 代表が兼任する場合は不要。規模拡大時に採用 |

## 2. 募集チャネル戦略

### チャネル一覧と優先度

| チャネル | 対象ポジション | 期待効果 | コスト | 優先度 |
|---------|--------------|----------|--------|--------|
| **X（Twitter）** | 全ポジション | ★★★ Claude Code / GCPコミュニティにリーチ | 無料 | A |
| **LinkedIn** | メイン講師、GCP TA | ★★★ 実務経験者に直接リーチ | 無料〜（InMail: ¥500/通） | A |
| **Connpass / TECH PLAY** | GCP TA | ★★☆ 勉強会参加者にリーチ | 無料（イベント掲載） | A |
| **Facebook（AI/開発系グループ）** | 全ポジション | ★★☆ コミュニティ内口コミ | 無料 | B |
| **Discord（AI/Claude系サーバー）** | 導入コースTA | ★★☆ Claude Code使いが集まる場所 | 無料 | B |
| **知人紹介（リファラル）** | メイン講師 | ★★★ 質が最も高い | 紹介料 ¥30,000〜50,000 | A |
| **Wantedly** | 全ポジション | ★★☆ 副業人材にリーチ | 月額 ¥40,000〜 | C |
| **クラウドワークス / ランサーズ** | 運営スタッフ | ★☆☆ 技術人材には不向き | 手数料20% | C |
| **GCP User Group（GCPUG）** | GCPメイン講師、GCP TA | ★★★ GCP専門家が集まる場 | 無料 | A |

### チャネル別の施策詳細

#### X（Twitter）— 最重要チャネル

**施策:**
1. メイン投稿（パターンA: 詳細版）を代表アカウントから投稿
2. 共感型（パターンB）でリポスト＆引用投稿
3. 問いかけ型（パターンD）でエンゲージメントを獲得し、興味を持った人にDMでアプローチ
4. #ClaudeCode #バイブコーディング #GCP #GoogleCloud 等のハッシュタグを活用

**投稿スケジュール:**
| 日 | 内容 | 目的 |
|----|------|------|
| Day 0（月曜） | パターンA メイン投稿 | 情報を網羅的に伝える |
| Day 1（火曜） | パターンD 問いかけ型 | エンゲージメント獲得 |
| Day 3（木曜） | パターンB 共感型をリポスト | リーチ拡大 |
| Day 5（土曜） | 「残り◯日」のリマインド | 締切効果 |
| Day 7（日曜） | 締切当日の最終告知 | 駆け込み応募 |

#### LinkedIn — GCP人材向け

**施策:**
1. 日本語の記事投稿で本格運用コースの魅力を伝える
2. GCPエンジニア、クラウドアーキテクト等のキーワードで検索→InMailで直接スカウト
3. 「Google Cloud」「Firebase」関連グループに投稿

**投稿テンプレート（LinkedIn用）:**
```
【サポート講師募集】歯科×Google Cloud の新しい挑戦

D-Code Academyでは、歯科医院の先生方に
Google Cloudを使った本格的なアプリ開発を教える
「本格運用コース」を開講します。

Cloud Run / Cloud SQL / Firebase Auth の
使用経験がある方で、「教える」ことに興味がある方、
ぜひご応募ください。

報酬：27.5万円/期（TA）、50-60万円/期（メイン講師）
形式：完全リモート（Zoom）・日曜のみ

#GoogleCloud #CloudRun #教育 #副業
```

#### GCPUG（Google Cloud User Group）— GCP専門家向け

**施策:**
1. GCPUGのSlack / Discordで募集情報を共有
2. 勉強会での登壇 or LT（Lightning Talk）で認知を獲得
3. GCPUGのイベントに参加して直接声をかける

#### Connpass / TECH PLAY — 技術者コミュニティ

**施策:**
1. 「D-Code Academy 説明会」をイベントとして登録
2. AI、Claude Code、GCP関連のイベント参加者にリーチ
3. イベント参加後に個別にアプローチ

書類選考（フォーム回答を評価）

オンライン面談（Zoom 15-30分）

採用通知（メール）

### リスク対策

| リスク | 対策 |
|--------|------|
| GCP経験者が集まらない | ① 報酬を上方調整（+20%まで） ② GCPUGでの直接スカウト ③ 導入コースTAを育成してGCPコースに昇格 |
| 応募数が不足 | ① Wantedly / YOUTRUST 等の有料チャネルを追加 ② 紹介料を設定してリファラルを強化 |
| 採用後の辞退 | ① 契約時に前金（報酬の20%）を支払い ② 補欠採用者を1-2名キープ |

## 5. 応募フォームの構成

### 共通項目（全ポジション）
1. 氏名（必須）
2. メールアドレス（必須）
3. SNSアカウント（任意）
4. 応募ポジション（必須・選択式）
   - 導入コース サポート講師（TA）
   - 本格運用コース サポート講師（TA）
   - 本格運用コース メイン講師
   - 運営スタッフ
   - 複数ポジションに興味がある
5. Claude Codeの使用頻度（必須・選択式）
6. Claude Codeの利用形態（必須・複数選択）
7. Claude Codeで作ったもの（必須・自由記述）

### 導入コースTA向け追加項目
8. Next.js / Reactの経験（選択式）
9. Vercel / Supabaseの使用経験（各選択式）

### 本格運用コースTA・メイン講師向け追加項目
8. GCPの使用経験（選択式）
   - 実務で3年以上使用
   - 実務で1〜3年使用
   - 個人開発やハンズオンで使用
   - ほぼ未経験
9. 使用したことのあるGCPサービス（複数選択）
   - Cloud Run
   - Cloud SQL
   - Cloud Storage
   - Firebase Auth / Firebase全般
   - Cloud Functions
   - Cloud Build
   - Cloud Monitoring / Logging
   - BigQuery
   - その他
10. Docker / コンテナの経験（選択式）
11. CI/CDの構築経験（選択式）

### 共通追加項目（全ポジション）
12. 教える・サポートの経験（任意・自由記述）
13. 初心者サポートで大切にしていること（必須・自由記述）
14. 参加可否（必須・選択式）
15. 志望動機（必須・自由記述）
16. ポートフォリオ / GitHub URL（任意）
