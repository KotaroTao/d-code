# サポート人員要件一覧

## 全体構成

D-Code Academyでは「導入コース」と「本格運用コース」の2コースを運営するため、
各コースに適したスキルセットを持つサポート人員を配置する。

---

## 人員体制

### 1コホートあたりの必要人数

| ポジション | 導入コース | 本格運用コース | 合計 |
|-----------|-----------|---------------|------|
| メイン講師 | 1名 | 1名 | 2名 |
| サポート講師（TA） | 3〜4名 | 2〜3名 | 5〜7名 |
| 運営スタッフ | 1名（兼任可） | 1名（兼任可） | 1〜2名 |
| **合計** | **5〜6名** | **4〜5名** | **8〜11名** |

### 受講者対サポート講師の比率
- 導入コース：受講者5〜7名に対しTA 1名（初心者が多いため手厚く）
- 本格運用コース：受講者5〜7名に対しTA 1名（技術的に複雑なため）

---

## ポジション別 詳細要件

### A. メイン講師

#### 導入コース メイン講師

| 項目 | 内容 |
|------|------|
| 役割 | 講義の進行、ライブデモ、全体のファシリテーション |
| 必須スキル | ① Claude Code WEB版の熟練使用 ② Next.js / React の実務経験 ③ Vercel + Supabase でのデプロイ・運用経験 ④ 初心者向けの説明力（専門用語を使わない解説） ⑤ Zoomでの講義・ファシリテーション経験 |
| 歓迎スキル | ・ 教育・研修の企画経験 ・ 歯科業界の知識 ・ UI/UXデザインの知見 |
| 稼働時間 | セミナー 15h + 準備 20h + 受講者対応 15h = 約50h/コホート |
| 報酬目安 | ¥400,000〜500,000/コホート（税別） |

#### 本格運用コース メイン講師

| 項目 | 内容 |
|------|------|
| 役割 | GCP講義の進行、アーキテクチャ設計のガイド、ライブデモ |
| 必須スキル | ① Claude Codeの日常的使用 ② Google Cloud の実務経験（Cloud Run, Cloud SQL, Firebase Auth, Cloud Storage のいずれか3つ以上） ③ CI/CD パイプラインの構築経験 ④ インフラ設計・運用の知見 ⑤ 初心者〜中級者への技術指導経験 |
| 歓迎スキル | ・ GCP認定資格（Associate Cloud Engineer以上） ・ Docker / コンテナの深い理解 ・ セキュリティの知見（IAM、暗号化） ・ 歯科業界の知識 |
| 稼働時間 | セミナー 15h + 準備 25h + 受講者対応 15h = 約55h/コホート |
| 報酬目安 | ¥500,000〜600,000/コホート（税別） |

---

### B. サポート講師（TA）

#### 導入コース TA

| 項目 | 内容 |
|------|------|
| 役割 | ブレイクアウトルームでの個別サポート、エラー解決支援、質問対応 |
| 必須スキル | ① Claude Codeを日常的に使っている（最重要） ② Next.js / React の基本的な理解 ③ 日本語でのコミュニケーション力 ④ 日曜 10:00-13:00 × 5週連続参加 |
| 歓迎スキル | ・ 教える・サポートの経験（塾講師、メンター等） ・ Supabase / Vercel の使用経験 ・ 歯科業界の知識 |
| 不要なもの | ・ 高度なプログラミングスキル ・ 講師経験 |
| 稼働時間 | セミナー 15h + セミナー後対応 20h + 準備 10h = 約45h/コホート |
| 報酬 | ¥225,000/コホート（税別） |

#### 本格運用コース TA

| 項目 | 内容 |
|------|------|
| 役割 | GCP操作の個別サポート、環境構築支援、トラブルシューティング |
| 必須スキル | ① Claude Codeの日常的使用 ② Google Cloud の使用経験（Cloud Run または Cloud SQL の基本操作ができる） ③ Docker / コンテナの基本理解 ④ GitHub Actions の基本理解 ⑤ 日曜 10:00-13:00 × 5週連続参加 |
| 歓迎スキル | ・ GCPの本番運用経験 ・ CI/CDパイプラインの構築経験 ・ Firebase Authの使用経験 ・ インフラ系の資格 ・ 教える・サポートの経験 |
| 不要なもの | ・ GCP認定資格（あれば嬉しいが必須ではない） |
| 稼働時間 | セミナー 15h + セミナー後対応 20h + 準備 15h = 約50h/コホート |
| 報酬 | ¥275,000/コホート（税別） |
